# Marketplace locale (monorepo)

Ce dossier regroupe les différentes briques du projet :

- `apps/mobile` : application mobile (Flutter)
- `apps/admin-web` : dashboard web d’administration
- `apps/api` : backend API
- `packages` : code partagé (types, utilitaires)
- `docs` : documentation (cahier des charges, schémas, etc.)
- `infra` : infrastructure (docker, déploiement, etc.)
- `scripts` : scripts (seed, migrations, maintenance)

## Backlog (User stories) & priorisation

### MVP (V1 – 4 semaines)
#### Auth & profils
- **Acheteur** : je peux créer un compte et me connecter.
- **Vendeur** : je peux créer un compte vendeur et compléter mon profil (contacts, activité).
- **Système** : je peux gérer des rôles (acheteur, vendeur, admin principal, admin secondaire).

#### Données de base
- **Admin** : je peux gérer catégories (CRUD).
- **Admin** : je peux gérer villes/régions (CRUD).

#### Annonces
- **Vendeur** : je peux créer/modifier/supprimer une annonce avec photos, prix, description, catégorie.
- **Vendeur** : je peux activer/désactiver une annonce (disponible / indisponible).
- **Acheteur** : je peux consulter une annonce et voir les infos clés (localisation, contacts).

#### Recherche & découverte
- **Acheteur** : je peux rechercher par mot-clé.
- **Acheteur** : je peux filtrer par catégorie et distance (ou ville/quartier).
- **Acheteur** : je peux trier (plus proche, plus récent).

#### Mise en relation
- **Acheteur** : je peux contacter le vendeur via WhatsApp (lien direct).
- **Acheteur** : je peux appeler / envoyer un email depuis l’annonce.

#### Abonnements (follow)
- **Acheteur** : je peux suivre/désuivre un vendeur.
- **Acheteur abonné** : je reçois une notification push quand un vendeur suivi publie une annonce.

#### Carte & géolocalisation (version MVP)
- **Acheteur** : je peux voir des vendeurs/annonces sur une carte.
- **Système** : je peux calculer la distance approximative (GPS).
- **Vendeur** : je peux définir **un point principal** (où me trouver) avec des horaires.

#### Modération “progressive”
- **Acheteur** : je peux signaler une annonce.
- **Admin** : je vois les signalements et je peux retirer l’annonce / avertir / suspendre.

### V2 (post-V1)
- Messagerie interne.
- Promotions avancées (codes promo, périodes, mise en avant payante éventuelle).
- Professions/secteurs, tags, SEO local.
- Reviews/notes (avec anti-abus).
- Analytics vendeur (vues, clics WhatsApp).

### V3
- Commande/livraison/paiement (si souhaité).
- Recommandations personnalisées.
- Système anti-fraude plus poussé + trust score.

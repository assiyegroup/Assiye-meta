## Portée du projet (V1 – cible 4 semaines)

### Contexte
Marketplace locale géolocalisée au Togo pour **mettre en relation** acheteurs et vendeurs, via une **app mobile Flutter** et un **dashboard admin Next.js**.

### Objectif V1 (MVP)
- Permettre à un vendeur de **publier des annonces** (produits/services) avec sa localisation.
- Permettre à un acheteur de **rechercher** et **trouver** des annonces proches + contacter rapidement (WhatsApp / appel / email).
- Permettre à l’admin de **structurer** (catégories, villes/régions) et **modérer** (signalements, désactivation, contrôles).

### Hors-scope V1
- Paiements / commandes intégrées (mise en relation uniquement).
- Livraison et suivi logistique complet (à cadrer en V2 si besoin).
- Recommandations avancées / IA / personnalisation poussée.

### Contraintes
- Délai V1 : **4 semaines minimum**.
- Mobile : **Flutter**.
- Dashboard : **Next.js**.
- Backend : API (proposition : **NestJS**) + base de données (proposition : **PostgreSQL**).

### Hypothèses V1 (à valider)
- Authentification par **téléphone** ou **email** (choix à trancher selon le contexte local).
- Les annonces sont **publiées immédiatement** mais soumises à une **modération progressive** (signalements + contrôles + restrictions).
